# UAC-System-Capacity-Analytics
Data analytics and AI-based forecasting project to analyze system capacity and care load for Unaccompanied Children (UAC) using Python, EDA, and Prophet.

# Project Overview
This project analyzes the system capacity and care load for Unaccompanied Children (UAC) using data analytics and AI-based forecasting techniques.
The goal is to understand system pressure, monitor inflow vs outflow, and predict future load using time-series modeling.

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Key Metrics:
  - Total System Load
  - Net Intake
- Data Visualization (Matplotlib)
- AI Forecasting using Prophet
- Interactive Dashboard using Streamlit

## Project Structure
- `eda.py` → Data analysis and forecasting
- `app.py` → Streamlit dashboard
- `uac_dataset.csv` → Dataset
- `forecast_output.csv` → AI prediction output
- `.png files` → Graphs and visualizations

## AI Implementation
Implemented time-series forecasting using Prophet to predict future system load and support decision-making.
